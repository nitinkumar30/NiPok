Arre bhai, ye hai **NiPok** (Nitin + Poker = NiPok) — aakhirkar ek poker app jo **Hindustani hai, dil se hai, aur INR mein hai!** No more of that foreign "chips" nonsense. Yahan **₹** (Rupaiya) chalta hai. Poora paisa vasool.

Koi signup nahi, koi Google login nahi, koi "forgot password" ki bakwas. Bas naam daalo aur khelo. Yeh hai asli **Aatmanirbhar Poker**. 💪🇮🇳

## 🛠️ Installation (Dev Kaise Karein?)

### Prerequisites (Pehle Se Laga Lena Bhai)

- [Node.js](https://nodejs.org/) v18+ (nahi toh update karo, 2025 bhog liya)
- npm ya yarn (jo bhi aapka mazhab hai)

### Clone Karke Chalo

```bash
# Project kholo
cd "C:\Users\Nitin Kumar\.opencode\bin\poker-platform"

# Sab package install karo (chai nahi milegi sorry)
npm install
cd server && npm install && cd ..
cd client && npm install && cd ..

# Prisma database banao
cd server && npx prisma db push && cd ..

# Chala do!
npm run dev
```

Ye command dono **frontend (port 5173)** aur **backend (port 3001)** ek saath chalayegi. Terminal mein 2 windows khulengi — ek client ka, ek server ka. Dono chalu hain toh browser mein **http://localhost:5173** pe jao.

## 🌍 Sports Bhai Bhai Hain (Multiplayer Kaise Kaam Karta Hai)

Desi Poker real-time multi-player hai. Kaam karta hai yeh:

1. **Server** (Node.js Express Socket.IO) — sab ka daddy. Cards bantata hai, rules enforce karta hai, validation karta hai
2. **Client** (React) — tumhara mobile/laptop. Bas dikhta hai aur action bhejta hai
3. **Connection** — Socket.IO real-time connection, jaise chai tapri pe WiFi free hai

> **Architecture:** Server authoritative. Matlab server pe aankh band karke bharosa. Client pe trust mat kar. Hum client ko kuch nahi bataate jo secret hai — woh India mein hain, lekin "authoritative server" ka logic America se liya hai (sorry, lekin code aacha hai).

## 💸 INR Currency — Why?

### Kyonki Hum Indian Hain!

1. **Dollar chhod do** — $100 lagta hai ₹10,000 jaisa? Bas psychological warfare hai!
2. **₹ me socho** — ₹10,000 ka pot hai toh pata chalega ki kitna kharcha hua
3. **Indian number system** — Lakh aur Crore mein sochna hai. Thousand nahi
4. **Aatmanirbhar** — Local currency, local game, local mazaa

> **Fun Fact:** Poker mein jab aap ₹1000 lagate ho, toh aap 🍛 ₹500 ki thali aur ₹500 ka gulab jamun miss kar rahe ho. Maje lijiye lekin control mein rahiye. Yeh sirf game hai.
